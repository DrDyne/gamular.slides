$resource - Usage


Class
  • get
  • save
  • query
  • remove
  • delete
Instance
  • $save
  • $remove
  • $delete

HTTP GET "class" actions: Resource.action(...)

non-GET "class" actions: Resource.action(...)

non-GET instance actions: instance.$action(...)

Class have access to all actions

Instance only have access to non-GET actions (no query/get/...)
